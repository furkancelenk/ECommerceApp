GGTechApp-ECommerce
</h1>

## Project Features
 - MVVM with Clean Architecture
 - Navigation Components
 - Retrofit
 - Firebase Auth & Firestore
 - Room Database
 - Glide
 - Lottie (<a href="https://github.com/LottieFiles/lottie-android">Source</a>) <br>
 - ViewPager2
 - Speech to Text (Google)
